# PortfolioApp
회원가입 / 자산배분 / 주식 목록 화면으로 이루어져 있는 자산관리 포트폴리오 앱

## 개발 환경
- Xcode 16.1
- macOS 15.1.1
- 최소지원버전 iOS 16
